# eMAXIS Slim オルカン 参考推計

eMAXIS Slim 全世界株式(オール・カントリー)の直近基準価額と、午前10時・午後6時の参考推計を前日比で表示する静的Webアプリです。

## 使い方

GitHub Pagesでこのリポジトリの `main` ブランチ、`/root` を公開してください。

アプリ本体は次の静的ファイルだけで動きます。

- `index.html`
- `styles.css`
- `app.js`
- `data/snapshot.json`
- `icon.svg`
- `site.webmanifest`

## データ更新

別サーバーは使いません。GitHub Actionsが `data/snapshot.json` を日本時間の午前10時台・午後6時台・午後11時台に更新してコミットします。

外部サイトの一時的な 500 や接続失敗が起きることがあるため、更新スクリプトは 15秒 → 30秒 → 60秒 の順で自動リトライします。手動更新したい場合は、GitHubのActionsタブから `Update forecast snapshot` を実行してください。

## 推計方法

直近の基準価額に、ACWI ETF(米ドル建て)の日次変化率とドル円の日次変化率を単純加算して掛けています。

これは正式な基準価額ではありません。実際の採用為替、評価タイミング、組入銘柄差、ETF固有要因、信託報酬、配当、時差、休日差は未調整です。

目安として推計値の±1%程度ずれる可能性があります。
